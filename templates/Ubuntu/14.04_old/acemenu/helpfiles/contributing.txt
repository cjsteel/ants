Contributing
============
If you are interested in contributing see our github project:

    "ansible-role-acemenu"

* create a clone of the project
* create a (short) help file and name it using a single word followed by .txt
* add your help file to the helpfiles directory.
* append the title (name without .txt) to the menuitems.j2 template(s).
* make a pull request.

Subjects might include; useful commands, cheat sheets, concise overviews... 

Regards,

The ACElabs IT, Research and Development Teams
